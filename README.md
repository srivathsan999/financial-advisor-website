# FinAdvisors - Financial Advisor Website Template

A premium, corporate-grade HTML5 template designed for financial advisors, wealth management firms, and investment consultancies.

## Project Structure
financial-advisor-website/
├── index.html          # Main Home Page
├── home-2.html         # Alternative Home Page
├── services.html       # Services Listing
├── service-details.html# Single Service Detail
├── case-studies.html   # Client Success Stories
├── about.html          # About Us / Team
├── pricing.html        # Pricing Plans
├── blog.html           # Blog Listing
├── blog-details.html   # Single Blog Post
├── contact.html        # Contact Form
├── login.html          # Client Login
├── signup.html         # Client Signup
├── 404.html            # Error Page
└── assets/             # CSS, JS, Images, Fonts

## Features
- **Design System:** Professional Blue/Slate color palette with Glassmorphism accents.
- **Dark Mode:** Fully supported via `theme-toggle.js` and `localStorage`.
- **Animations:** Smooth GSAP fade-up and stagger effects.
- **Responsive:** Built on Bootstrap 5 grid for mobile-first layout.
- **Images:** Sourced from Unsplash (placeholders included).

## Setup
1. Open the project folder in VS Code.
2. Use "Live Server" extension to view `index.html`.
3. No build process required (Static HTML/CSS/JS).

## Customization
- **Colors:** Edit `--primary-color` and other variables in `assets/css/style.css`.
- **Dark Mode:** Edit variables in `assets/css/dark.css`.
- **Content:** Update HTML files with real text.
